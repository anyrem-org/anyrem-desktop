## Dashboard / Memory Hub

The dashboard should not be a generic analytics page. It should provide real user value by helping the user remember, continue, review, and search.

The dashboard should answer:

- What did I capture today?
- Where did I stop?
- What should I review?
- What topics am I focusing on?
- What can I search or reopen quickly?

MVP dashboard sections:

- Large search box
- Today’s captured notes
- Continue where you left off
- Recently active notes
- Top categories today/this week
- Daily recap preview

Avoid useless vanity metrics in MVP:

- Total notes all time
- Too many charts
- Complex analytics
- Large graph on dashboard
- AI insights before data quality is good

The dashboard should feel like a “Memory Hub”, not an admin analytics page.
